## Library Management API
![LibraryAPI](https://socialify.git.ci/msizi007/LibraryAPI/image?language=1&name=1&owner=1&stargazers=1&theme=Light)

A professional RESTful API developed with **Node.js**, **Express**, and **TypeScript**. This project provides a structured backend for managing books and authors, featuring robust request validation and automated development workflows.

---

#### How to run project

1. clone the project from github using:

```bash
git clone https://github.com/msizi007/LibraryAPI.git
```

2. install all node dependancies

```bash
npm install
```

3. use npm run dev to run the project

```bash
npm run dev
```

---

### Features

- **Full CRUD Operations**: Complete management for `Books` and `Authors`.
- **TypeScript Integration**: Type-safe development using `ts-node`.
- **Data Validation**: Integrated `express-validator` to ensure IDs are strictly integers.
- **Hot Reloading**: Automated server restarts using `nodemon`.
- **Modular Architecture**: Separated concerns between Routers, Controllers, and Server configuration.

---

### Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: TypeScript
- **Development Tools**: Nodemon, ts-node
- **Validation**: Express-Validator
