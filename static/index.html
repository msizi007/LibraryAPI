Library API Documentation

Use the endpoints below to interact with the Library Management System. Ensure Content-Type: application/json is set for POST/PUT requests.

📚 Books Management

Method Endpoint Description Validation / Notes
GET /books Retrieve all books -
GET /books/:id Get specific book by ID ID must be an integer
POST /books Create a new book entry Requires JSON body
PUT /books/:id Update book credentials ID must be an integer
DELETE /books/:id Remove book from library ID must be an integer
